# Amlak Ultra (Android)

این پروژه یک اپ اندروید (WebView) است که فایل HTML شما را داخل خودش اجرا می‌کند.

## خروجی APK با GitHub Actions
1) ریپو را در GitHub بسازید و کل فایل‌های این پروژه را داخلش آپلود کنید.
2) تب **Actions** → Workflow **Build APK (Debug)** را اجرا کنید.
3) بعد از موفقیت، پایین صفحه‌ی Run بخش **Artifacts** فایل `app-debug-apk` را دانلود کنید.
